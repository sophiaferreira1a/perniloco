# Perniloco

Perniloco é um jogo desenvolvido em Python com Pygame onde o jogador controla um mosquito tentando evitar obstáculos em ambientes domésticos.

## Sobre o jogo

O jogador controla o Perniloco, um pernilongo que voa pela tela enquanto sofre efeito da gravidade.

Ao pressionar a barra de espaço ou clicar na tela, o jogador faz o personagem subir. O desafio é desviar de pares de raquetes elétricas que surgem pelo caminho.

A dificuldade aumenta conforme o jogador avança pelas fases.


## Objetivo

Sobreviver desviando dos obstáculos e completar todas as fases sem perder todas as vidas.


## Mecânicas principais

- Gravidade constante puxando o personagem para baixo  
- Impulso de subida ao pressionar espaço ou clicar  
- Sistema de 3 vidas por fase  
- Colisões com obstáculos, chão ou teto causam perda de vida  
- Avanço de fase ao atingir pontuação mínima  

## Controles

- `Espaço` → subir
- `Clique do mouse` → subir
- `Enter` → iniciar/reiniciar o jogo


## Recursos utilizados

- Imagens autorais produzidas com a ferramenta LibreSprite
- Fonte estilo pixel
- Referências de mecânica baseadas em Flappy Bird


## Dificuldades previstas

- Implementar física de gravidade de forma fluida  
- Detecção precisa de colisões  
- Transições entre fases sem travamentos  



## Escopo mínimo

- Personagem controlável  
- Obstáculos em movimento  
- Sistema de vidas  
- Sistema de pontuação   


## Melhorias futuras

- Telas de início e fim de jogo
- Sons e música de fundo
